- 运行 bin\Debug 下的 ConversionToolsBox.exe 即可。
- 16位表示法可以是带 # 或不带 # 的。
- RGB 和 HSL 的三个参数请使用英文逗号 ',' 分割。
- HSL 请使用浮点数表示明度和饱和度。

- Run the ConversionToolsBox.exe in the bin\Debug directory to start.
- The 16-bit representation can be with or without the “#” symbol.
- Please separate the three parameters of RGB and HSL using a comma “,” in English.
- For HSL, use floating-point numbers to represent luminance and saturation.
